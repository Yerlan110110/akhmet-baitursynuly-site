# Шоқан Уәлиханов туралы веб-сайт

Бұл жоба практикалық жұмысқа арналған. Тақырыбы: **2-студент. Шоқан Уәлиханов - қазақтың тұңғыш ғалымы**.

## Бөлімдер

- Өмірі
- Ғылыми зерттеулері
- Саяхаттары
- Еңбектері
- Галерея

## Қолданылған технологиялар

- HTML
- CSS
- JavaScript

## Қосымша функционал

- Мобильді мәзір
- Dark/light mode
- Модаль терезе
- Жоғарыға көтерілу батырмасы
- Smooth scroll

## Іске қосу

`index.html` файлын браузерде ашыңыз.
