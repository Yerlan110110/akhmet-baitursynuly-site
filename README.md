# Ахмет Байтұрсынұлы туралы веб-сайт

Бұл жоба практикалық жұмысқа арналған. Тақырыбы: **3-студент. Ахмет Байтұрсынұлы - қазақ тілінің реформаторы**.

## Бөлімдер

- Өмірбаяны
- Әліпби реформасы
- Әдеби шығармалары
- Ағартушылық қызметі
- Галерея

## Қолданылған технологиялар

- HTML
- CSS
- JavaScript

## Қосымша функционал

- Мобильді мәзір
- Dark/light mode
- Модаль терезе
- Жоғарыға көтерілу батырмасы
- Smooth scroll

## Іске қосу

`index.html` файлын браузерде ашыңыз.
